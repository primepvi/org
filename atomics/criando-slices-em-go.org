#+title: Criando Slices em Go
#+created_at: <2025-11-13 qui>
#+edited_at: [2025-11-13 qui]
#+tags: :programação :golang

Para criarmos *slices* em *go* podemos usar a *sintaxe* com *valores literais*, essa maneira é bem semelhante como inicializamos *arrays*, entretanto sem especificar o *tamanho*:

#+begin_src go
slice := []int{5, 10, 15, 20, 25}
#+end_src

Por baixo dos panos o *compilador* cria um *array* com *tamanho* e *capacidade* *iguais* e faz o *slice* *apontar* para ele, caso queiramos definir nós mesmos o *tamanho* e *capacidade* podemos usar a *função* make da seguinte forma:

#+begin_src go
/* 
Utilizando make para criar slices com tamanho e capacidade pré-definidos:
make([]int, 5) -> len=5, cap=5
make([]int, 2, 4) -> len=2, cap=4 
*/

slice := make([]int, 3)
#+end_src 

Quando utilizamos o make os primeiros *len* elementos do *array* recebem o valor padrão do *tipo especificado* para os *elementos* do *array.*
